CSS Temel Ödevi

Ana Sayfa Hakkında Ilgilendiklerim İletişim
Verilen benzer bir düzenle basit bir web sayfası oluşturun. CSS'in temel özelliklerini (display, width, height, margin, padding, border, selector) uygulayın. Sadece HTML5 + CSS3 kullanın, hazır framework kullanmak yasak.

İçerik

Bu alanda örnek içerik metinleri bulunmaktadır. CSS özellikleriyle metinler arasında boşluklar (line-height) ve hizalamalar (text-align) düzenlenir.

Önce CSS ile kutuları / alanları yan yana getirmeyi, ardından alt alta yerleştirmeyi öğrendik.

Yan Menü / Bilgi

Bu alan yan menü veya bilgi kısmıdır. CSS ile genişlik (width), yükseklik (height), kenar boşlukları (margin/padding) ve kenarlık (border) özellikleri kullanılarak düzenlenebilir.
